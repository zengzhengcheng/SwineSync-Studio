# SwineSync Studio v1.0.0

Included release assets:

- `main.exe`
- `trabase_model.onnx.enc`
- `tragan_model.onnx.enc`
- `traganbase_model.onnx.enc`
- `first.obj`

The empty placeholder file `初始模型.obj` is stored in the repository at:

- `release_layout/userModels/初始模型.obj`

Usage notes:

- Download all release assets.
- Download `release_layout/userModels/初始模型.obj` from the repository.
- Place `main.exe` at the root folder.
- Place the three `.enc` model files in a `models/` subfolder.
- Place `first.obj` and `初始模型.obj` in a `userModels/` subfolder.
- The first run may be blocked by antivirus software.
- The software requires network access on the first run and once every 10 launches.
- Once the software has been run, do not move its filesystem location.
- If you move it, re-download the software.
- If you need multiple parallel copies, download separate copies into different filesystem locations.
- This build stops working after `2027-05-05`.
- The expiration date will be updated whenever a new version is released.
- This is intended to encourage users to keep using the latest version of the software, because new releases will continue to include new features, improvements, and fixes.
